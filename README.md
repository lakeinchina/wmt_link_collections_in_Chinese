https://github.com/weimingtom/RuleTransitionDemo  

* 缓动公式（Tween，Easing，补间动画）  
(1) 说明：  
来源于Flash的概念，一切浮点数属性皆可缓动（汗）

https://github.com/weimingtom/FireTreeFrame

https://code.google.com/p/tweener/

http://robertpenner.com/easing/
